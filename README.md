# Python_GUI_apps
GUI Python apps

## Dependencies
- scopefoundry
- numpy
- pyqt
- qtpy
- h5py
- pyqtgraph
- matplotlib
- scipy
- lmfit
- customplotting

## Installation
```
conda install numpy pyqt qtpy h5py pyqtgraph
pip install git+git://github.com/ScopeFoundry/ScopeFoundry.git
pip install matplotlib scipy lmfit customplotting==0.1.4.dev0
```
